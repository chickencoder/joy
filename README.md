Joy
---

Joy is an extremely lightweight command line interface which organises and renders
a simple static sites.

#### Install
<!-- ```bash
npm install -g joycli
``` -->
<!-- 
or... -->

```bash
git clone https://github.com/chickencoder/joy.git
cd joy
npm install -g
```
